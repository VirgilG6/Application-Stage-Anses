## Installation
1. Créer un espace de stockage sur votre ordinateur (exemple: suivi_de_projets):
```
cd suivi_de_projets
```

2. Cloner le projet en utilisant la commande suivante: 
```
git clone https://github.com/VirgilG6/Application-Stage-Anses.git
```
